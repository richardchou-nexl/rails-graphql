Rails + React + TypeScript + esbuild. Use it for interview or your application.

Ruby 3.2.7

Rails 8.0.2

Has been tested on Windows WSL and Mac.

## Setup

```
bundle install
yarn install
rails s
```

## TypeScript watch mode

```
yarn build --watch
```

## Tests

```
rspec
```

## Rubocop

```
rubocop --autocorrect-all
```
